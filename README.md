# SQL Minor Project: Bike Store & Airbnb Analysis

## Project Overview
This project focuses on practicing core SQL concepts using JOINs and aggregation functions. The analysis is performed on two datasets: Bike Store Sales and Airbnb Booking data.

## Bike Store Analysis
Tasks performed:
- Store revenue analysis
- Top 5 most sold products
- Customers and their store orders
- Total products per brand
- Products with zero stock

## Airbnb Analysis
Tasks performed:
- Superhosts and total bookings
- Revenue by property type
- Average booking price by age group
- Total bookings by nationality
- Properties with rating above 4.5

## SQL Concepts Used
- INNER JOIN
- GROUP BY
- SUM(), COUNT(), AVG()
- WHERE
- ORDER BY

## Tools Used
- SQL
- Relational Database

## Goal
To strengthen SQL fundamentals by solving real-world business problems using relational queries.
